# SHOPEZ - E-Commerce Website

## 📌 Project Overview
SHOPEZ is a full-stack e-commerce web application that allows users to browse products, add them to the cart, and place orders. The project demonstrates both frontend and backend development.

## 🚀 Features
- User-friendly interface
- Product listing
- Shopping cart
- Checkout page
- Responsive design
- Backend API
- MongoDB database integration

## 🛠️ Technologies Used

### Frontend
- React.js
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

## 📂 Project Structure

```
SHOPEZ
│
├── frontend
├── backend
└── project planning
```

## ▶️ How to Run

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
